Simple REST API Instructions

  • Create a new user

  • To see the existing users type "/api/user"

  • To see only one user type "/api/user/[username]"

  • To see only one attribute of a user type "/api/user/[username]?[attribute]"

    Examples:
    "/api/user/[username]?name"
    "/api/user/[username]?username&birthdate"

  • To update a user type "/api/user/[username]/update"

  • To delete a user type "/api/user/[username]/delete"
